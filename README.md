## Usage

```bash
git clone <this-repo>
npm install

# terminal 1
npx hardhat node

# terminal 2
npx compile
npx hardhat hardhat run scripts/sample-script.js --network localhost

# terminal 3 
npm run dev
```

- Update the deployed contract address in `compoonents/Home.js`
